fun labelOffsetPart(labelArg: String, dbg: DebugInfo): Int

Return the numeric offset part of LABELARG, where LABELARG may be either (result 0), + (result as an Int), -, +, or -. Here, refers to a .equiv'ed symbol that resolves to an immediate constant (as in .equiv value, 13).
